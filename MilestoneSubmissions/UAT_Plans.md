**UAT Plans**

UAT 1: Update feature
Test Data Description: The user will enter a valid location name, address, and coordinate pair and see it update on the map.
Test Environment: The test will be done on localhost, where the site will run for all users.
Test Results: The user must see the location they submitted on the map and in the location cards.
Tester Information: The user must never have accessed the site before and must be unfamiliar with the site specifically.

UAT 2: Register feature
Test Data Description: The user will enter a valid registration credentials including a username, password, and email address.
Test Environment: The test will be done on localhost, where the site will run for all users.
Test Results: The user must be redirected to the login screen where they can use the same credentials to successfully log in.
Tester Information: The user must never have accessed the site before and must be unfamiliar with the site specifically.

UAT 3: Review feature
Test Data Description: The user must be able to enter a review and post it to the site.
Test Environment: The test will be done on localhost, where the site will run for all users.
Test Results: The user must see the review they submitted attached to the location.
Tester Information: The user must have used the home page and understand the location card feature.
